# Alert, Nunavut, Canada

## Basic information
**GNSS station ID**:        [ALRT](https://webapp.geod.nrcan.gc.ca/geod/data-donnees/station/report-rapport.php?id=M029001)            
**Location**:               82.494, -62.340 <br/>
**Biome**:                  Polar desert <br/>
**MAAT**:                   -18 degC <br/>
